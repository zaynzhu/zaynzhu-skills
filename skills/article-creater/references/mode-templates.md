# 各模式结构模板与自检标准

> 本文件定义 article-creater（公众号写作）四种写作模式的结构模板、自检标准和特殊规则。

---

## 一、长文模式

### 触发词

写文章、写稿子、公众号长文、出稿、长文、帮我写

### 字数

4000-8000 字

### 完整流程

```
素材 → 热点搜索 → 选题判断 → 大纲生成 → 用户确认大纲 → 写作 → 快扫+终审 → 输出
```

### 大纲模板

生成大纲时必须包含以下要素：

```
【文章原型】调查实验型/产品体验型/现象解读型/工具分享型/方法论分享型
【开头方式】叙事启动/荒诞事实/热点破题/好奇心驱动
【风格锦囊】选定 3-5 个（从风格锦囊列表中选择）
【段落规划】
  - 开头：[具体切入方式]
  - 段落 1：[核心观点] — [素材支撑]
  - 段落 2：[核心观点] — [素材支撑]
  - ...
  - 升维：[从哪个点 → 连接到哪个文化/哲学/历史参照物]
  - 收尾：[结尾类型]
【回环呼应】开头埋 [xxx] 钩子，结尾用 [xxx变体] callback
```

### 自检标准

**快扫（10 项）**：

| # | 检查项 | 通过标准 |
|---|--------|---------|
| 1 | 禁用词 | 0 命中 |
| 2 | 禁用标点 | 0 命中（冒号、破折号、双引号） |
| 3 | 套话模式 | 0 命中 |
| 4 | 空泛工具名 | 0 命中 |
| 5 | 开头是否具体切入 | 不是宏大叙事开头 |
| 6 | 长短句交替 | 连续 3 句以上句式长度相近 = 不通过 |
| 7 | 口语化词组 | 至少 8 个不同表达 |
| 8 | 一句话独立成段 | 全文至少 3 次 |
| 9 | 扣主线句 | 每个偏离主线的段落后都有 |
| 10 | 小标题 | 除方法论型外不应出现 |

**终审（6 维度）**：

1. 每个核心观点都有具体支撑吗？（人/场景/细节/数据）
2. 知识是"随手掏出来"的还是"下面我来科普"的？
3. 有没有文化升维？连接是否自然？
4. 对立面有理解和承认吗？
5. 文章类型专项检查通过吗？
6. 活人感判断：温度感、独特性、姿态、心流——有 AI 味重的段落吗？

## 二、短内容模式

### 触发词

小红书、朋友圈、短推文、短文案、短内容

### 字数

200-800 字

### 流程

```
素材 → 热点搜索（简化版）→ 选题判断 → 直接写作 → 快扫(1-7项) + 终审简化版 → 输出
```

**砍掉**：大纲步骤、文化升维、人物画像法、英雄之旅叙事弧
**保留**：核心锚点（活人语气、具体不空泛、绝对禁区）、口语化词组、开头必杀技中的"好奇心驱动"和"荒诞事实"

### 结构模板

```
【钩子】一个让人想看下去的具体场景/事实/疑问（1-2 句）
【核心观点】你真正想说的那件事（2-5 句）
【收尾】行动呼吁 / 金句收尾（1 句）
```

### 自检标准

**快扫（1-7 项）**：

| # | 检查项 | 通过标准 |
|---|--------|---------|
| 1 | 禁用词 | 0 命中 |
| 2 | 禁用标点 | 0 命中 |
| 3 | 套话模式 | 0 命中 |
| 4 | 空泛工具名 | 0 命中 |
| 5 | 开头是否具体切入 | 不是宏大叙事开头 |
| 6 | 长短句交替 | 连续 3 句以上句式长度相近 = 不通过 |
| 7 | 口语化词组 | 至少 3 个不同表达 |

**终审简化版**：
- 活人感：读起来像真人说的吗？
- 具体性：有具体细节/数据/人名吗？

## 三、续写模式

### 触发词

续写、接着写、扩写、补全

### 字数

续写部分 1000-3000 字

### 流程

```
已有文本 → 风格指纹提取 → 大纲生成(续写部分) → 写作 → 快扫+终审+风格衔接检查 → 输出
```

### 风格指纹提取模板

读取用户提供的已有文本，提取以下信息：

```
【节奏】长短句比例：[ ] 句子平均长度：[偏短/中等/偏长]
【人称】第一人称/第二人称/混合
【标点习惯】是否使用。。。、？？？、= = 等：[ ] 冒号/破折号使用情况：[ ]
【情绪基调】[冷静理性/激动热血/温暖治愈/吐槽调侃]
【口头禅】[提取已有的口语化表达]
```

续写时必须延续以上风格指纹。大纲步骤中，续写部分的开头要能自然衔接原文的最后一段。

### 自检标准

与长文模式相同（快扫 10 项 + 终审 6 维度），外加：

**风格衔接检查**：
- 续写部分的第一段和原文最后一段的语气是否一致？
- 续写部分的人称、标点习惯、口头禅是否与原文匹配？
- 从原文到续写部分的过渡是否自然？读者能不能感觉到断点？

## 四、改写模式

### 触发词

用我的风格写、改写、改成我的风格、重写

### 字数

看原文长度，通常 4000-8000 字

### 流程

```
原文 → 骨架提取 → 大纲生成 → 写作 → 快扫+终审 → 输出
```

### 骨架提取模板

```
【核心观点】[原文想表达的 1-3 个核心观点]
【关键素材】[原文中的具体案例、数据、人名、事件]
【可丢弃】[原文的语言风格、组织方式、过渡句——这些会被改写]
【需保留】[原文中的核心事实和引述——改写时需要保留，但用自己的方式重新表述]
```

提取骨架后，按照长文模式的完整流程重新写：大纲 → 写作 → 快扫+终审。

### 自检标准

与长文模式相同（快扫 10 项 + 终审 6 维度），外加：

**忠实度检查**：
- 原文的核心观点是否完整保留？
- 原文中的关键素材（数据、案例）是否准确引用？
- 改写后是否有原文没有的新观点？（如果有，需要标注是新增）
