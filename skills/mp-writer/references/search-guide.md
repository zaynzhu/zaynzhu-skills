# 分类型搜索指南

> 本指南帮助 mp-writer 根据不同文章类型，在热点搜索阶段搜索正确的信息维度。

## 通用搜索原则

1. **先获取当前日期时间**，确保搜索最新信息
2. 搜索结果必须标注来源和时效性
3. 不确定或过时的事实不用
4. 如果搜索不到足够素材，主动告知用户，请求更多输入

## 按文章类型搜索

### 调查实验型

搜索重点：事件时间线、当事人信息、数据统计、对比实验结果

推荐搜索词组合：
- "{主题} 最新进展 {当前年月}"
- "{主题} 时间线 经过"
- "{主题} 数据 统计 调查"
- "{主题} 对比 实验 测试"

重点关注：
- 事件发生的具体时间节点和因果链
- 当事人/机构的背景信息
- 可量化的数据和统计结果
- 他人的实验或测试结果，用于对比验证

### 产品体验型

搜索重点：产品最新动态、竞品对比、用户真实评价、价格信息

推荐搜索词组合：
- "{产品名} 最新更新 {当前年月}"
- "{产品名} vs {竞品} 对比"
- "{产品名} 评测 体验 真实"
- "{产品名} 价格 订阅"

重点关注：
- 产品最近一次更新的具体功能变化
- 真实用户的使用感受（不是官方宣传）
- 竞品的优势和劣势对比
- 定价策略和性价比

### 现象解读型

搜索重点：现象起源、不同立场观点、学术研究、历史类比

推荐搜索词组合：
- "{现象} 起源 原因 怎么回事"
- "{现象} 观点 争议 正方 反方"
- "{现象} 研究 论文 学术"
- "{现象} 历史 类比 先例"

重点关注：
- 现象首次出现的时间和触发事件
- 不同立场的声音，尤其是对立面观点
- 学术研究的具体结论和论文引用
- 历史上的类似事件和结局

### 工具分享型

搜索重点：工具最新版本、使用教程、用户反馈、替代品

推荐搜索词组合：
- "{工具名} 最新版本 更新"
- "{工具名} 教程 使用方法"
- "{工具名} 评价 值不值"
- "{工具名} 替代 免费 类似"

重点关注：
- 工具当前版本号和最新功能
- 现有教程中的常见坑和解决方案
- 真实用户反馈中的高频问题和优点
- 市面上的替代品对比

### 方法论分享型

搜索重点：行业最新实践、学术支撑、常见失败案例、数据佐证

推荐搜索词组合：
- "{方法论关键词} 最佳实践 案例 {当前年月}"
- "{方法论关键词} 研究 论证 数据"
- "{方法论关键词} 失败 踩坑 常见错误"
- "{方法论关键词} 学习曲线 入门 难度"

重点关注：
- 行业内最新的实践案例和效果数据
- 学术领域的支撑性研究和结论
- 真实的失败案例和常见误区
- 学习成本和入门难度的具体数据

## 搜索结果提取框架

无论哪种文章类型，搜索后都要提取四类素材：

1. **肉**：具体事件和细节（时间、人物、数据）——让文章有料
2. **张力**：不同观点和争议——让文章有冲突感
3. **魂**：文化升维素材（历史事件、哲学概念、文学引用）——让文章有深度
4. **证**：可引用的数据和来源——让文章可信
