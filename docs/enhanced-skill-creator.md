# enhanced-skill-creator

Skill 全生命周期管理工具。

## 功能

从零创建、编辑优化、测试评测、打包交付——覆盖 Skill 从想法到成品的完整流程。

- 需求收集 → 草稿编写 → 自审 → 测试运行 → 评审反馈 → 迭代改进 → 描述优化 → 打包交付
- 内置 5 类技能模板库
- 多层次测试（L1 语法检查 / L2 逻辑验证 / L3 集成测试）
- 描述优化脚本，自动调整触发词精准度

## 依赖

- 无必需工具（纯指令型 Skill）
- 可选：Python（用于描述优化脚本）

## 使用方式

直接告诉 AI 你想做什么：

```
帮我做个技能，用来……
把这个流程保存成技能
优化一下我的 skill
```

Skill 会自动判断你处于哪个阶段，从那里接手。

## 文件结构

```
enhanced-skill-creator/
├── SKILL.md          ← 主指令
├── agents/           ← 子 agent 指令
├── assets/           ← 模板资源
└── references/       ← 参考文档
```
